В файле type-breaker-game.tsx находится код игры в стиле Arcanoid с механикой для развития навыка слепой печати. 

Сделай на основе кода полноценное  приложение React, которое может собираться в набор статических файлов. Во всех местах в коде переименуй "Arcanoid" на "Type Breaker". Используй TypeScript и vite для сборки. Создай package.json с именем проекта type-breaker-game, добавь все необходимые пакеты для сборки и разработки и установи их. В package.json  должны быть скрипты:   
dev - для сборки проекта в режиме разработке;
build - для сборки в виде статических файлов (html, css, js) в директорию dist;
typecheck - для проверки корректности типов.    
                                
Убедись, что проверка типов и сборка выполняются без ошибок сборка выполняются без ошибок
